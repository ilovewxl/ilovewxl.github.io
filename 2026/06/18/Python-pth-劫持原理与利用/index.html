Python.pth 劫持原理与利用

2026-06-18 Python · 权限提升 · 内网渗透

什么是 .pth 文件

Python 启动时会自动加载 site-packages 目录下所有 .pth 文件，把文件中的路径添加到 sys.path。

这样 /tmp/my_module 就会被加入模块搜索路径。
