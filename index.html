Python.pth 劫持原理与利用

什么是 .pth 文件

Python 启动时会自动加载 site-packages 目录下所有 .pth 文件，把文件中的路径添加到 sys.path。

这样 /tmp/my_module 就会被加入模块搜索路径。

劫持原理

Python 导入 site 模块时会逐行执行 .pth 中的内容，以 import 开头的行会被当成代码执行。

利用场景

场景一：写文件权限拿下 Python 环境

场景二：配合文件上传漏洞
